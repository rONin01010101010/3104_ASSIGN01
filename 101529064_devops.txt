COMP 3104 is a DevOps course that covers essential tools and practices used in modern software development. The curriculum includes Git for version control, Docker for containerization, and Jenkins for automation. Lab exercises focus on building CI/CD pipelines and deploying applications in real-world scenarios. The course also covers GitHub Actions and Kubernetes, which are frequently required skills in the tech industry. The comprehensive material demonstrates why DevOps knowledge is valuable for career advancement in software development.
