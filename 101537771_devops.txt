COMP 3104 - DevOps is a comprehensive course designed to introduce students to the principles and practices of DevOps methodology in software development and IT operations. This course is part of the Computer Systems Technology program at George Brown College.

This course covers essential topics including version control with Git and GitHub, continuous integration and continuous deployment (CI/CD) pipelines, automation, containerization with Docker, and infrastructure as code. Students learn to use industry-standard tools such as GitHub Actions, Jenkins, and various automation frameworks.

The curriculum emphasizes collaborative development practices, branching strategies, pull request workflows, and code review processes. Students gain hands-on experience with automated testing, deployment pipelines, and monitoring tools that are critical in modern software development environments.

Key learning outcomes include understanding the DevOps culture, implementing CI/CD workflows, managing infrastructure through code, and applying best practices for collaboration and automation in software projects. The course prepares students for real-world scenarios where development and operations teams work together to deliver high-quality software efficiently.

Throughout the course, students work on practical projects and assignments that simulate professional DevOps workflows, including repository management, automated builds, testing frameworks, and deployment strategies to cloud platforms.
